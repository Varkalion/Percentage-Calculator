# Percentage-Calculator

🇬🇧 English Version

Instant Online Percentage Change Calculator

A sleek, lightweight, and fast web-based tool designed to calculate the percentage difference between two values and apply percentage adjustments to any base number instantly.

Key Features:

    Real-time Calculation: Results appear instantly as you type.

    High Precision: Displays changes with up to 2 decimal places.

    Fully Responsive: Optimized for a seamless experience on mobile, tablets, and desktops.

    Clean Typography: Utilizes modern system fonts for maximum readability.

    User-Friendly UI: Built with Tailwind CSS, featuring intuitive color-coded results (Green for growth, Red for decline).

How to Use:
Simply enter your initial and final values. The tool automatically determines the trend and provides the exact percentage shift along with the absolute difference. You can also use the second section to quickly calculate markups or discounts on a specific amount.

نسخه فارسی (Persian Version)

محاسبه‌گر آنلاین و آنی درصد تغییرات

یک ابزار تحت وب ساده، سریع و کاربردی برای محاسبه درصد افزایش یا کاهش بین دو عدد و اعمال درصد بر روی مقادیر دلخواه. این پروژه با تمرکز بر سرعت کاربری و طراحی مدرن ساخته شده است.

ویژگی‌های کلیدی:

    محاسبه آنی: نتایج به محض تایپ کردن اعداد نمایش داده می‌شوند.

    دقت بالا: نمایش تغییرات تا ۲ رقم اعشار برای دقت بیشتر.

    طراحی واکنش‌گرا (Responsive): نمایش بی‌نقص در موبایل، تبلت و دسکتاپ.

    اعداد فارسی: پشتیبانی کامل از فونت وزیر و اعداد خوانای فارسی.

    رابط کاربری مدرن: استفاده از Tailwind CSS برای تجربه کاربری لذت‌بخش.

نحوه استفاده:
فقط کافیست مقادیر خود را در کادرها وارد کنید. ابزار به طور خودکار تشخیص می‌دهد که مقدار شما افزایش یافته یا کاهش، و میزان دقیق تغییر را با رنگ‌های استاندارد (سبز برای سود/افزایش و قرمز برای ضرر/کاهش) به شما نشان می‌دهد.
